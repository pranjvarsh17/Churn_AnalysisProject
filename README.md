# Customer Churn Analysis and Customer Intelligence

A complete end-to-end data analytics project focused on understanding customer churn through data cleaning, feature engineering, SQL analysis, and interactive visualizations.

The objective of this project is to transform raw customer data into meaningful business insights that can help organizations identify customer behavior, improve retention strategies, and support data-driven decision making.

---

## Project Overview

In this project, I worked with customer data stored in a SQLite database and performed the complete analytics workflow using Python.

The project covers:

- Importing data from a SQLite database
- Data cleaning and preprocessing
- Feature engineering
- SQL-based analysis
- Exploratory Data Analysis (EDA)
- Business insights through visualizations
- Customer intelligence generation

## Project Workflow

### 1. Data Import

- Connected to a SQLite database
- Loaded multiple tables into Pandas DataFrames
- Inspected schema and table structures

---

### 2. Data Cleaning

Performed several preprocessing tasks including:

- Renaming columns
- Removing unnecessary features
- Handling missing values
- Data type conversion
- Standardizing categorical values
- Data validation

---

### 3. Feature Engineering

Created meaningful features to improve analysis and business understanding.

Examples include:

- Customer age calculation
- Derived business metrics
- Table joins using customer identifiers

---

### 4. Exploratory Data Analysis

Analyzed customer behaviour to answer business questions such as:

- Which customers are more likely to churn?
- How do demographics affect customer retention?
- Which customer segments contribute the most?
- Distribution of customer characteristics
- Trend and pattern analysis

---

### 5. SQL Analysis

Used SQL queries to extract business insights from the database.

Worked with:

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- JOINS

---

### 6. Data Visualization

Created visualizations using Matplotlib and Seaborn to communicate findings effectively.

Visualizations include:

- Bar Charts
- Histograms
- Count Plots
- Pie Charts
- Distribution Plots
- Correlation Analysis
